# StudentsGo
**StudentsGo** is an innovative Python-based application designed to streamline transportation decision-making for individuals in Taipei City, particularly around National Taiwan University (NTU). By integrating data from MRT, buses, and YouBike, the program provides users with a one-stop solution to compare travel options based on estimated time, cost, and eco-friendliness. The application also features travel history recording for easy reference and insights into past trips.

Built with a user-friendly command-line interface, **StudentsGo** uses reliable methods like the Haversine formula for precise distance calculations and predefined transfer points for accurate MRT route suggestions. The system covers major destinations in Taipei City, enabling users to choose the most efficient and cost-effective mode of transport effortlessly. By consolidating multiple transportation options into a single interface, **StudentsGo** enhances the user experience, saving time and effort, and setting the stage for future expansions with real-time data and graphical interfaces.
